The biggest issue I ran into was, since I sorted all of my data into a 2D array of nearly identical Tkinter.Button objects, and all Button objects require a callback function on instantiation, I had no way to distinguish each button from eachother, they all called the same function.
To fix this I created a dynamic callback function that accepted an int as a parameter and used lambda functions from within the button instantiation which allowed me to instantiate buttons with identical callback functions, but unique function parameters.
